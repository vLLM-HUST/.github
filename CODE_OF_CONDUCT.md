# vLLM-HUST Code of Conduct

## Our Pledge

We want `vLLM-HUST` to be a practical, respectful, and technically serious open-source community.
Contributors, maintainers, and users should be able to collaborate without harassment, hostility, or exclusion.

We welcome participation regardless of age, body size, disability, ethnicity, sex characteristics, gender identity and expression, level of experience, education, socioeconomic status, nationality, personal appearance, race, caste, color, religion, or sexual identity and orientation.

## Our Standards

Examples of behavior that help the community:

- being respectful in technical disagreement
- giving actionable, specific feedback instead of personal criticism
- assuming good intent while still holding quality bars high
- reporting bugs, regressions, and security issues responsibly
- helping keep discussions focused, reproducible, and useful to others

Examples of unacceptable behavior:

- harassment, insults, or discriminatory language
- personal attacks, intimidation, or doxxing
- publishing private credentials, logs, or personal data without permission
- derailing technical threads into abusive or inflammatory arguments
- repeated bad-faith interactions after maintainers ask to stop

## Scope

This Code of Conduct applies across the `vLLM-HUST` GitHub organization, including repositories, issues, pull requests, discussions, release notes, and other community spaces.

It also applies when someone is representing the organization in public or semi-public contexts.

## Reporting

If you need to report unacceptable behavior, contact the maintainers privately through the security/contact path listed in the relevant repository, or open a private maintainer contact through the organization when available.

For urgent security-sensitive incidents, use the security reporting path described in [SECURITY.md](SECURITY.md).

## Enforcement

Maintainers may remove content, lock conversations, reject contributions, or restrict participation when behavior violates this Code of Conduct.

We will try to act proportionally, document serious actions when appropriate, and protect reporters from retaliation.
